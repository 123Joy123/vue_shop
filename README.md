# EWShop

## 项目介绍

EWShop是一款移动版图书商城项目，包含首页门户、商品详情、商品分类、购物车、订单提交、个人中心等模块。（个人练习项目，不可做商业用途）

## 所用技术

* Vue3
* Vue CLI, Vue Router, Vuex
* Vant
* Axios
